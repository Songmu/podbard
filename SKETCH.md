# SKETCH

## Directory Structure

- index.md - index page
- primcast.yaml - configuration file
- episode/ - eqisode pages
- audio/   - audio files
- images/  - images (optional)

## Sub Commmands

### init

```console
$ primcast init
```

### episode

```
$ primcast episode [-slug hoge] audio/1.mp3
```

create a new epoisode page with the specified audio file.

### build

```
$ primcast build
```
