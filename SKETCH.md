

# directory structure

- index.md - index page
- primcast.yaml - configuration file
- episode/ - eqisode pages
- audio/   - audio files
- images/  - images



# sub commmands

## init

```console
$ primcast init
```

## page

```
$ primcast episode audio/1.mp3
```

create a new epoisode page with the specified audio file.


## build

```
$ primcast build
```
